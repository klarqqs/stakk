## üõ†Ô∏è How to Capture Screenshots

### iOS (Xcode Simulator)

1. **Open Xcode**
2. **Run your app** in simulator
3. **Select device**:
   - iPhone 14 Pro Max (for 6.7")
   - iPhone 11 Pro Max (for 6.5")
   - iPhone 8 Plus (for 5.5")
4. **Navigate to desired screen**
5. **Take screenshot**:
   - `Cmd + S` or
   - Device ‚Üí Screenshot
6. **Screenshots saved to**: `~/Desktop`

### Android (Android Studio Emulator)

1. **Open Android Studio**
2. **Run your app** in emulator
3. **Select device**:
   - Pixel 6 Pro (for large phones)
   - Pixel 5 (for standard phones)
4. **Navigate to desired screen**
5. **Take screenshot**:
   - Click camera icon in toolbar, or
   - Extended Controls ‚Üí Screenshot
6. **Screenshots saved** to specified location

### Physical Devices

#### iOS
1. Navigate to screen
2. Take screenshot (Power + Volume Up)
3. Screenshots in Photos app
4. Export at full resolution

#### Android
1. Navigate to screen
2. Take screenshot (Power + Volume Down, or gesture)
3. Screenshots in Gallery
4. Export at full resolution

## üé® Screenshot Enhancement Tips

### Do's ‚úÖ
- Use **realistic test data** (not placeholder text)
- Show **actual app features** working
- Maintain **consistent theme** (all light or all dark)
- Use **high-quality images** (no blur, pixelation)
- Show **key value propositions** clearly
- Include **captions** if helpful (optional)

### Don'ts ‚ùå
- Don't use **placeholder content**
- Don't show **real personal information**
- Don't include **sensitive data** (real account numbers, etc.)
- Don't mix **light and dark themes** in same set
- Don't use **low-resolution images**
- Don't show **error states** or bugs

## üìê Screenshot Specifications

### File Format
- **PNG**: Best quality, larger file size
- **JPEG**: Smaller file size, good quality
- **Recommendation**: PNG for best results

### Color Space
- **sRGB** (standard RGB)
- Avoid CMYK or other color spaces

### Compression
- **No compression artifacts**
- Use high-quality settings
- Max file size: 500 MB (iOS), 8 MB (Android)
